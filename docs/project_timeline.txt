T2 — Factor of Safety Setup
  Once T1 shares the master feature table (Day 5),
  you will calculate the Factor of Safety column.
  For now, set up the notebook and write the formula
  as a Python function ready to apply later:

  import numpy as np

  def factor_of_safety(c, gamma, gamma_w, m, z, beta_deg, phi_deg):
      beta = np.radians(beta_deg)
      phi  = np.radians(phi_deg)
      num  = c + (gamma - gamma_w * m) * z * (np.cos(beta)**2) * np.tan(phi)
      den  = gamma * z * np.sin(beta) * np.cos(beta)
      return num / den

  Where:
    c       = competency_index * 20
    gamma   = 20
    gamma_w = 9.81
    m       = rainfall scaled to 0 to 1 range
    z       = 2.0
    beta    = slope column (degrees)
    phi     = 30 + competency_index * 10

PROBLEM STATEMENT — COPY THIS INTO SUBMISSION FORM
================================================================

"Landslide susceptibility in the Garhwal Himalaya is governed
by both surface morphology and subsurface rock competency.
Existing models rely solely on remote sensing-derived features
such as slope, NDVI and curvature, ignoring subsurface
lithological controls on slope stability. This project
introduces a Lithological Proxy Transfer Framework that uses
wireline well log data (GR, RHOB, NPHI) to learn a
geomechanical rock competency relationship and maps it onto
Himalayan lithological units via regional geology data from
GSI Bhukosh. Combined with satellite-derived terrain and
vegetation indices from Google Earth Engine and a physics-
informed Factor of Safety feature from the Infinite Slope
Equation, this multi-source feature set is used to train a
landslide susceptibility classifier for the MCT zone near
Helang-Joshimath. The pipeline is designed as a globally
transferable framework applicable to any mountainous region
by substituting regional bounding coordinates and lithology
lookup tables."
